# 🚀 Инструкция по запуску Music Sync Room

## Шаг 1: Установка PostgreSQL

### Вариант A: Docker (рекомендуется)
1. Установите Docker Desktop: https://www.docker.com/products/docker-desktop/
2. Запустите Docker Desktop
3. Откройте терминал в папке `back-end`:
   ```bash
   cd back-end
   docker compose up -d
   ```
4. База данных будет доступна на `localhost:5432`

### Вариант B: Локальная установка PostgreSQL
1. Скачайте и установите PostgreSQL: https://www.postgresql.org/download/
2. Во время установки запомните пароль пользователя `postgres`
3. Откройте pgAdmin или psql и выполните:

```sql
-- Создать базу данных
CREATE DATABASE collab_db;

-- Создать пользователя
CREATE USER collab_user WITH PASSWORD 'collab_pass';

-- Дать права
GRANT ALL PRIVILEGES ON DATABASE collab_db TO collab_user;

-- Подключиться к базе и дать права на схему
\c collab_db
GRANT ALL ON SCHEMA public TO collab_user;
```

4. Инициализируйте базу данных:
   - Откройте pgAdmin
   - Подключитесь к базе `collab_db`
   - Откройте Query Tool
   - Скопируйте содержимое файла `back-end/server/db/init/001_init.sql`
   - Выполните скрипт

## Шаг 2: Настройка Яндекс OAuth

1. Перейдите на https://oauth.yandex.ru/client/new
2. Авторизуйтесь в Яндекс аккаунте
3. Заполните данные приложения:
   - **Название**: Music Sync Room (или любое другое)
   - **Платформа**: Web-сервисы
   - **Redirect URI**: `http://localhost:3000/auth/callback`
   
4. Выберите права доступа (нажмите "Выбрать права"):
   - ✅ `login:info` — доступ к информации пользователя
   - ✅ `login:avatar` — доступ к аватару пользователя
   - ✅ `playlist:read` — чтение плейлистов (для будущего функционала)
   - ✅ `playlist:write` — добавление треков в плейлист

5. Нажмите "Сохранить"
6. Скопируйте:
   - **Client ID** (ID приложения)
   - **Client Secret** (Секретный ключ)

## Шаг 3: Настройка бэкенда

1. Откройте терминал в папке `back-end/server`:
   ```bash
   cd back-end/server
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Откройте файл `.env` в папке `back-end/server` и замените:
   ```
   YANDEX_CLIENT_ID=ваш_client_id_из_шага_2
   YANDEX_CLIENT_SECRET=ваш_client_secret_из_шага_2
   ```

## Шаг 4: Запуск бэкенда

1. В терминале `back-end/server`:
   ```bash
   npm run dev
   ```

2. Вы должны увидеть:
   ```
   ✅ Подключение к PostgreSQL установлено
   🚀 Server running on port 5000
   📡 Health check: http://localhost:5000/health
   ```

## Шаг 5: Запуск фронтенда

### Вариант A: Live Server (рекомендуется для разработки)
1. Откройте VS Code
2. Установите расширение "Live Server"
3. Откройте файл `front-end/public/index.html`
4. Нажмите правой кнопкой → "Open with Live Server"

### Вариант B: Простой HTTP сервер
1. Откройте новый терминал в папке `front-end/public`:
   ```bash
   cd front-end/public
   npx http-server -p 3000
   ```

### Вариант C: Открыть файл напрямую (не рекомендуется)
Откройте `front-end/public/index.html` в браузере (некоторые функции могут не работать из-за CORS)

## Шаг 6: Проверка работы

1. Откройте браузер: `http://localhost:3000`
2. Нажмите "Войти через Яндекс"
3. Разрешите доступ приложению
4. После авторизации:
   - Введите имя (опционально)
   - Нажмите "Создать комнату"
   - Скопируйте код комнаты
5. Откройте новую вкладку браузера
6. Войдите под другим аккаунтом (или тем же)
7. Нажмите "Войти по коду" и введите код комнаты

## 🔍 Проверка функциональности

### ✅ Поиск треков
1. Введите название трека или исполнителя в поле поиска
2. Нажмите "Найти"
3. Должны появиться результаты из Яндекс Музыки

### ✅ Добавление в очередь
1. Нажмите "+ В очередь" у найденного трека
2. Трек должен появиться в списке очереди
3. У других участников в комнате очередь обновится автоматически

### ✅ Голосование
1. Нажмите 👍 или 👎 у трека в очереди
2. Score трека должен измениться
3. Очередь должна пересортироваться по score

### ✅ Воспроизведение
1. Ведущий (HOST) нажимает "▶ Play"
2. У всех участников должно отобразиться "⏸ Pause"
3. Нажмите "⏭ Next" для перехода к следующему треку

### ✅ Сохранение трека
1. Когда трек играет, нажмите "💾 Сохранить в плейлист"
2. Трек должен добавиться в "Мне нравится" Яндекс Музыки

## 🐛 Решение проблем

### Ошибка: "Не удалось подключиться к базе данных"
- Убедитесь, что PostgreSQL запущен
- Проверьте параметры в `.env`:
  ```
  DB_HOST=localhost
  DB_PORT=5432
  DB_NAME=collab_db
  DB_USER=collab_user
  DB_PASSWORD=collab_pass
  ```

### Ошибка: "Требуется повторная авторизация"
- Токен Яндекс истёк
- Выйдите из приложения и войдите снова

### Ошибка: "Socket.io не подключается"
- Убедитесь, что бэкенд запущен на порту 5000
- Проверьте консоль браузера на ошибки CORS

### Ошибка: "Redirect URI mismatch"
- Проверьте, что в настройках Яндекс OAuth указан точно:
  `http://localhost:3000/auth/callback`

### Frontend не видит бэкенд
- Проверьте, что в `script.js` указан правильный API_URL:
  ```javascript
  const API_URL = 'http://localhost:5000';
  ```
